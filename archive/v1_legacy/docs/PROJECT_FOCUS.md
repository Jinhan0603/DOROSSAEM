# Project Focus

## 지금 프로젝트의 핵심 목표

`강사관리 시스템 개선` 프로젝트의 현재 핵심 목표는 다음 하나다.

- 수업이 끝나면 최소 3개의 기록이 자동으로 남는 구조를 만든다

그 3개는 아래다.

1. 기관/담임 피드백
2. 학생 마이크로 피드백
3. 강사 회고

## 지금 사용자가 집중할 일

- 위 3개 폼의 문항 확정
- 각 폼에 반드시 들어갈 `request_id`, `instructor_id` 확인
- 수업 종료 후 누가 언제 폼을 받는지 운영 흐름 결정

## 지금 Codex가 맡을 일

- 시트/Apps Script 구조 유지
- 기존 Phase 2 폼 연결/점검 코드 유지
- 기록 원장을 연결할 데이터 구조 정리
- 이후 점수 집계와 배정 구조로 이어질 수 있게 스키마와 로직 유지

## 지금 하지 않아도 되는 일

- 증거팩 제출 폼의 상시 운영
- 고도화된 자동 배정 알고리즘
- DB 서버/웹앱 확장

## 컨텍스트 72%% 관련

- 72%%는 아직 작업 중단 상태가 아니다
- 대화 컨텍스트는 자동으로 압축될 수 있다
- 그래서 현재 프로젝트의 기준 문서는 저장소 안 문서로 유지한다

핵심 기준 문서:

1. `docs/PROJECT_FOCUS.md`
2. `docs/IMPLEMENTATION_CHECKLIST.md`
3. `docs/V2_PLAN.md`
4. `docs/SCORING_SPEC_V1.md`
