# Simplified Scoring Spec

## 1. 원칙

- 점수는 하나만 쓴다.
- 점수의 목적은 보상용 복잡한 평가가 아니라 `배치 판단`이다.
- 매니저가 설명할 수 없는 점수는 쓰지 않는다.

## 2. 최종 점수

`최종 점수 = 강의 참여 점수 + 강의 피드백 점수 + 기여 로그 점수 - 패널티 점수`

## 3. 점수 항목

### 3.1 강의 참여 점수

활동 로그에서 쌓는다.

예:

- 일반 강의 참여
- 행사 참여
- DLS 참여
- 운영인력 참여

### 3.2 강의 피드백 점수

아래 3종 기록에서 계산한다.

- 기관/담임 피드백
- 학생 마이크로 피드백
- 강사 회고

## 3.3 기여 로그 점수

강의 외 활동을 반영한다.

예:

- OT 참여
- 콘텐츠 교육 수강
- 교안 제작
- 교구재 제작
- OT 강의
- 교육 영상 제작
- 포장 알바
- 추천
- 월간 우수 선정

### 3.4 패널티 점수

감점은 누락 없이 누적한다.

예:

- 지각
- 당일 취소
- 노쇼
- 현장 문제

## 4. 역할 판단

점수는 역할 판단과 연결된다.

- `보조강사 배정`
- `주강사 배정`
- `DLS 주강사 배정`
- `운영인력 배정`

지표는 복잡하게 쪼개지 않는다.

핵심은:

- 강의 경험이 있는가
- 피드백이 안정적인가
- 추가 기여가 있는가
- 패널티가 없는가

## 5. 등급

- `General`
- `Advanced`
- `Master`
- `Penalty`

기본 해석:

- `General`: 기본 배치 가능
- `Advanced`: 주강사 안정권
- `Master`: 운영인력 포함 핵심 인재
- `Penalty`: 배치 제한

## 6. 중요한 운영 규칙

- 등급보다 역할 컬럼이 우선이다.
- 점수보다 패널티 적용 누락이 더 치명적이다.
- 모든 자동 계산은 `Activity_Log`와 피드백 로그를 근거로 한다.
