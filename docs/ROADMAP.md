# DORO 강사관리 시스템 로드맵

## 전체 목표

DORO 강사관리 시스템을 구축하여, 강사 역량 데이터를 기반으로  
**강의 배정 → 등급 관리 → 운영 자동화**를 단계적으로 달성합니다.

---

## 1단계: 기획/설계 (1개월 차)

> **현재 진행 중**

### 목표
- 강사 DB Ver 2 구조 확정 및 구축
- 핵심 데이터 시트 4개 완성
- HTML/JS 기반 관리 도구 구현

### 세부 항목
- [x] 등급 산정 기준표 확정 (General / Advanced / Master / Penalty)
- [x] DB 스키마 설계 (Instructor_DB, Instructor_Sensitive, Activity_Log, Ability_Log)
- [x] 강사 의견 수렴 (구글폼 기본 정보 등록)
- [ ] **강사 DB HTML 관리 도구 구축**
- [ ] **샘플 데이터 입력 및 검증**
- [ ] **GitHub 연동 및 초기 push**

### 다음 단계 연결
> 강사 DB 구축 완료 후 → **강의 일정 DB** + **강사 신청/배정 현황 프로그램** 제작

---
