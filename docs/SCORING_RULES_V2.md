# DORO 강사 점수/등급 규칙 Ver 2

## 1. 원칙

- 점수는 **총점 하나**만 쓴다.
- 점수의 목적은 **배치 판단**이다 (보상용 평가 아님).
- 매니저가 설명할 수 없는 점수는 쓰지 않는다.

---

## 2. 총점 계산

```
총점 = 강의 활동 점수 + 내부 기여 점수 + 추천/수상 점수 - 패널티 점수
```

각 항목은 `Activity_Log`와 `Ability_Log`에서 자동 집계됩니다.

---

## 4. 등급 체계

| 등급 | 조건 | 가능 역할 |
|------|------|----------|
| **General** | 이번 기수 활동 여부 = `active` | 보조강사 중심 |
| **Advanced** | 총점 75점 이상 | 일반 수업 주강사, 보조강사 |
| **Master** | 상위 20명 (매월 갱신) | DLS 주강사, 캠프/전체행사/부스 운영, 일반 수업 주강사, 보조강사 |
| **Penalty** | 패널티 누적 | 배정 불가 |

### 등급별 강사비

#### Master DOROSSAEM (마-쌤)
- 행사: 운영인력 20만, 주강사 35만, 보조 30만
- 일반: 운영인력 2만, 주강사 2.8만, 보조 1.5만

#### Advanced DOROSSAEM (어-쌤)
- 일반: 주강사 2.5만, 보조 1.2만

#### General DOROSSAEM (젠-쌤)
- 일반: 보조 1.2만

---

## 5. 운영 규칙

1. 등급보다 **역할 적합성**이 우선이다.
2. 점수보다 **패널티 누락 방지**가 더 중요하다.
3. 모든 자동 계산은 `Activity_Log`와 `Ability_Log`를 근거로 한다.
4. Master는 **매월 1일** 기준 상위 20명을 재산정한다.
